# Taskly — Frontend

Interfaz web para la aplicación de gestión de tareas Taskly.

## Stack
- React + Vite
- lucide-react (íconos)

## Requisitos
- Node.js 18+
- El backend corriendo en localhost:8080 → github.com/MatiasEzr/taskly-api

## Cómo correrlo
npm install
npm run dev

Abrí http://localhost:5173
